# 🚀 AmbitionBox Web Scraping & Data Cleaning Pipeline

## 📌 Overview

This project demonstrates an end-to-end web scraping and data preprocessing workflow using Python. It automates data collection from publicly accessible web pages, extracts structured information, and performs data cleaning to prepare the data for analysis.

> **Note:** This repository contains **source code only**. No data obtained from AmbitionBox is included.

---

## ✨ Features

- Automated multi-page web scraping
- HTML parsing using BeautifulSoup
- Structured data extraction
- Error handling and request validation
- Data cleaning and preprocessing
- Export to CSV format
- Modular and well-documented code

---

## 🛠️ Technologies Used

- Python 3.x
- Requests
- BeautifulSoup4
- Pandas
- NumPy
- lxml

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

1. Open `scraper.ipynb`
2. Run all cells to collect the raw data.
3. Open `cleaner.ipynb`
4. Run all cells to clean and preprocess the data.
5. Export the processed data for further analysis.
